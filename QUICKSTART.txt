FIRST TIME SETUP:
-----------------

1. Install Prerequisites:
   - Python 3.8+ (https://www.python.org/downloads/)
   - Node.js 16+ (https://nodejs.org/)

2. Setup Backend:
   cd backend
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt

3. Setup Frontend:
   cd frontend
   npm install

EVERY TIME YOU START:
--------------------

Option A - Use Quick Start Script (Easiest):
   Double-click: START_WINDOWS.bat

Option B - Manual Start:

   Terminal 1 (Backend):
   cd backend
   venv\Scripts\activate
   python app.py

   Terminal 2 (Frontend):
   cd frontend
   npm run dev

   Browser:
   Open http://localhost:3000

USING THE SYSTEM:
----------------

1. Start Treatment Cycle:
   - Click "Start Cycle" button
   - Watch phase change to FILLING
   - Water level chart updates in real-time

2. Monitor Progress:
   - Phase Timeline shows current stage
   - Component Status shows active pumps/blower
   - Event Log records all actions

3. Manual Control (when cycle not running):
   - Turn individual components ON/OFF
   - Test each pump and blower independently

4. Emergency Stop:
   - Red button stops everything immediately
   - Click "Reset Emergency Stop" to resume
